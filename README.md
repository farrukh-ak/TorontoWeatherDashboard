**Toronto Winter Analysis — Climate Data vs Personal Experience**
An end-to-end data project analysing whether the 2025–26 Toronto winter was objectively harsher than historical norms, motivated by lived experience as a newcomer to Canada.

**Skills demonstrated:**

**Data sourcing:** Kaggle dataset (1937–2025) + Meteostat Python API (2025–26)
**Data engineering:** Python · Pandas · API integration · multi-source merging · schema normalisation · handling NaN/mixed types
**Data modelling:** Power BI data model · DAX calculated columns · DAX measures · table relationships
**Data visualisation:** Power BI · matrix heatmap · conditional formatting · bookmark navigation · tooltip pages · sync slicers
**Storytelling:** Translating a personal observation into a data-backed narrative across a 4-page interactive report
**Tools:** Python · Pandas · Meteostat · Power BI Desktop · DAX
